# LitLink — Virtual Audio Cable for Mac

A virtual audio driver for macOS that routes system audio and microphone input to any application.

**[Download LitLink](https://github.com/linguaholic/litlink-app/releases/latest/download/LitLink.pkg)** · macOS 14+ · Signed & notarized by Apple

## Features

**Free:**
- System audio passthrough with automatic multi-output device
- Mic passthrough with volume control
- Built-in companion app with visual controls

**Pro ($3.99/mo or $29 lifetime):**
- Per-app audio routing
- Individual volume control per app
- Per-app mute and level meters
- Parametric EQ per app and master EQ
- Routing profiles

Learn more at [litpads.app/litlink](https://litpads.app/litlink)
